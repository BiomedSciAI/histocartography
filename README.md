# Installation 

## PyPI installer

`pip install histocartography`

## From sources 

- Create a conda environment:

```
conda env create -f environment.yml
```
## Tests

To ensure proper installation, run unit tests as:

```sh 
python3 -m unittest discover -v
```

- Activate it:

```
conda activate histocartography
```
